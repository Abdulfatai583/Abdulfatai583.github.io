Project Overview

This Power BI project analyzes call center operations to evaluate agent performance, call volume, resolution efficiency, and customer satisfaction. It helps managers improve service quality and operational decisions.

Dataset

  • Agent: Call center agent name
  • Call Duration: Duration of calls
  • Call Volume: Number of calls handled
  • Resolution Status: Resolved / Unresolved
  • Customer Satisfaction: CSAT rating
  • Date: Call date

Key Metrics

  • Agent performance analysis
  • Call resolution efficiency
  • Monthly call volume trends
  • Customer satisfaction insights
